## 简单绘图软件
# 概述
这是一个基于Open Cascade Technology (OCC) 构建的简单绘图软件，旨在提供基础的3D建模和绘图功能。

# 特点
3D建模：支持创建立方体、圆柱体、球体等基本几何形状。
几何运算：允许进行布尔运算、平移、旋转等几何操作。
交互式界面：提供直观易用的图形用户界面 (GUI)，方便查看和编辑模型。
# 依赖
Open Cascade Technology (OCC)（版本7.0或更高）
Qt5或Qt6（用于图形界面）

## 功能
本软件具备现有CAD软件的主要基础功能，包括：
基础绘图：支持绘制点、线段、圆、B样条曲线、长方体、球体、圆环体等图形。
基础图形设计：提供边离散点计算、两条边之间的极值点计算、点到线/面的投影、圆角、倒角、旋转、镜像、偏移等功能。

**项目初衷：本项目旨在探索OCC的基本用法并研究现有CAD软件的设计理念。虽然实用性有限，但我们欢迎任何形式的贡献！**
